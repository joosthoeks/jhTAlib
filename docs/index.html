Install

From PyPI:

$ [sudo] pip3 install jhtalib

From source:

$ git clone https://github.com/joosthoeks/jhTAlib.git
$ cd jhTAlib
$ [sudo] pip3 install -e .

Update

From PyPI:

$ [sudo] pip3 install --upgrade jhtalib

From source:

$ cd jhTAlib
$ git pull [upstream master]
